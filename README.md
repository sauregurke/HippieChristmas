**Hippie Christmas**, a curbside marketplace for discarded items, built with React Native and Firebase. WIP!

Dump that old recliner without the burden of communicating with people! Pop it onto the curb, share it with the world, and some broke college kid will have a **Hippie Christmas** when they haul it away.
